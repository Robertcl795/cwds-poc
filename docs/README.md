# Design System Documentation

This repo now uses two canonical documentation tracks.

- Track A: Architecture & Vision (`docs/architecture-vision`)

Anything outside these tracks is considered legacy implementation history and should not be used as source-of-truth for new contributors.

9. Review [Serious DS Roadmap (Task 5)](./architecture-vision/task5-serious-design-system-roadmap.md)
