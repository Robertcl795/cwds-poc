# Phase 1 Primitives

Implemented primitives:

- button
- icon-button
- checkbox
- radio
- switch
- divider
- progress

Key contracts:

- native-first semantics (`button`, `input`, `progress`, `hr`)
- optional ripple enhancement for action controls
- shared `data-*` state hooks for styling
- token-driven styling via `--cv-comp-*` aliases
