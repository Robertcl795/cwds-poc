# @ds/tokens

Status: `stable`

Purpose: the single source of truth for primitive and semantic design tokens.

What belongs here:
- primitive token definitions
- semantic token definitions
- theme-level token overrides
- token layer ordering

What does not belong here:
- runtime behavior
- base/reset CSS
- component styling

Public entry points:
- `@ds/tokens/index.css`
- `@ds/tokens/themes/color-scheme.css`
