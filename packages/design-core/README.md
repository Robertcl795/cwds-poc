# @ds/core

Status: `stable`

Purpose: concise runtime contracts for the design system.

What belongs here:
- interaction controllers and overlay infrastructure
- a11y/runtime helpers used by primitives and web components
- icon registration/runtime helpers
- small Lit-oriented attribute composition helpers for internal components

What does not belong here:
- token CSS
- component CSS
- styling layer ownership
- compatibility-only utility sprawl

Public entry points:
- `@ds/core`
- `@ds/core/lit`
