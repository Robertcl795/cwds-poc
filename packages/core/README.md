# @ds/core

Status: `stable`

Purpose: concise runtime contracts for the design system.

What belongs here:
- interaction controllers and overlay infrastructure
- a11y/runtime helpers used by components and web components
- icon registration/runtime helpers
- nonvisual shared helpers for fields, overlays, navigation, feedback, and web-component plumbing
- reset, theme, motion, and shared utility CSS

What does not belong here:
- token CSS
- component CSS
- styling layer ownership
- visual component implementations

Public entry points:
- `@ds/core`
- `@ds/core/icons`
- `@ds/core/index.css`
