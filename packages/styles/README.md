# @ds/styles

Status: `stable`

Purpose: the single source of truth for tokens and CSS layers.

What belongs here:
- design tokens
- reset, theme, motion, and shared foundation CSS
- component and adapter CSS
- CSS QA conventions

What does not belong here:
- runtime behavior
- Lit/controller logic
- duplicated token packages

Public entry points:
- `@ds/styles/tokens.css`
- `@ds/styles/index.css`
